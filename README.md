# BuildBuddy AI 🚀

**An AI-powered coding mentor that analyzes your code and teaches you how to fix it.**

---

## 📌 Overview

BuildBuddy AI is an intelligent development assistant designed to empower students and developers by providing insightful code analysis, debugging guidance, and conceptual explanations. Unlike conventional AI tools that deliver direct solutions, BuildBuddy AI emphasizes understanding and learning, turning errors into valuable educational moments.

**Key Highlights:**
- Analyze code snippets or development issues
- Receive context-aware explanations and debugging steps
- Get actionable suggestions for code improvements
- Enhance programming skills through guided learning

Built as part of the **AWS AI for Bharat Hackathon**, BuildBuddy AI aims to foster a deeper understanding of programming concepts while assisting in troubleshooting.

---

## 🎯 Problem Statement

Learning programming and modern tech stacks presents several challenges:
- Confusing or ambiguous error messages
- Time-consuming debugging process
- Reliance on copying solutions without comprehension
- Overwhelming documentation

Current AI tools often provide quick fixes but fall short in fostering genuine understanding, creating a gap between building code and mastering programming principles.

---

## 💡 Our Solution

BuildBuddy AI bridges this gap by acting as a **personal AI coding mentor** that:
- Provides **detailed, step-by-step explanations**
- Clarifies **why errors occur** and **how to fix them**
- Suggests **best practices** and **code improvements**
- Promotes **conceptual learning** alongside debugging

This approach transforms errors into learning opportunities, fostering long-term coding competency.

## ⚙️ System Workflow

1. **User Input:** Code snippet or error query
2. **Input Processing:** Cleans and prepares data
3. **AI Context Understanding:** Interprets user intent
4. **Code Analysis Engine:** Detects issues and patterns
5. **Error Detection:** Finds syntax/logical errors
6. **Explanation Generation:** Crafts educational responses
7. **AI Mentor Guidance:** Presents step-by-step assistance
8. **User Action:** Applies fixes or learns concepts

---

## 🏗 System Architecture

The platform integrates a frontend interface with AI-powered analysis modules:

- **Frontend:** HTML, CSS, JavaScript (Vite)
- **AI Integration:** OpenAI API, Google Gemini API
- **Backend / Processing:** Node.js, API calls for code understanding and explanation
- **Deployment:** Vercel

## 🔧 Installation & Setup

Follow these steps to run BuildBuddy AI locally:

1. **Clone the repository:**
```bash
git clone https://github.com/yourusername/buildbuddy-ai
```
2. **Navigate to project directory:**
```bash
cd buildbuddy-ai
```
3. **Install dependencies:**
```bash
npm install
```
4. **Start development server:**
```bash
npm run dev
```
5. **Open in your browser:**
[http://localhost:5173](http://localhost:5173)
